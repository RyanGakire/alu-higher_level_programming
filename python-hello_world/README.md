# python-hello_world


This project is an introduction to the Python programming language.
It covers writing and running basic Python scripts via shell wrappers,
printing strings, integers and floats with f-strings, string slicing
and concatenation, and basic string manipulation without loops or
conditional statements.

## Files
- `0-run`: Shell script that runs a Python script (`$PYFILE`)
- `1-run_inline`: Shell script that runs inline Python code (`$PYCODE`)
- `2-print.py`: Prints a fixed string using `print`
- `3-print_number.py`: Prints an integer followed by text using f-strings
- `4-print_float.py`: Prints a float with 2 decimal precision
- `5-print_string.py`: Prints a string 3 times and its first 9 characters
- `6-concat.py`: Concatenates two strings into a sentence
- `7-edges.py`: Demonstrates string slicing (first/last characters)
- `8-concat_edges.py`: Builds a new sentence using only slicing, no new literals
- `9-easter_egg.py`: Prints the Zen of Python
