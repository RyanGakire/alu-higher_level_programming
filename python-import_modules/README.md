# 0x02. Python - import & modules

## Description

This project covers how to import functions and variables from external files in Python, work with command-line arguments via `sys.argv`, and use built-in module introspection tools like `dir()`. It is part of the ALX Software Engineering Program's Python fundamentals track.

## Learning Objectives

- How to import functions and variables from another file
- How to use imported functions
- How to create a module
- How to use the built-in function `dir()`
- How to prevent code in a script from being executed when it is imported
- How to use command line arguments with Python programs

## Requirements

- Python 3.8.x
- All files are executable
- All files start with `#!/usr/bin/python3`
- Code follows `pycodestyle` (version 2.7.*)

## Files

| File | Description |
|------|-------------|
| `0-add.py` | Imports `add(a, b)` from `add_0.py` and prints the result of `1 + 2 = 3` |
| `1-calculation.py` | Imports `add`, `sub`, `mul`, `div` from `calculator_1.py` and prints results for `a = 10`, `b = 5` |
| `2-args.py` | Prints the number of arguments passed and lists them with their positions |
| `3-infinite_add.py` | Prints the sum of all command-line arguments; handles big numbers |
| `4-hidden_discovery.py` | Prints all non-dunder names defined in the compiled module `hidden_4.pyc`, in alphabetical order |
| `5-variable_load.py` | Imports the variable `a` from `variable_load_5.py` and prints its value |
