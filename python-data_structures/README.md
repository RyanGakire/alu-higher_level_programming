# Python List & Tuple Utilities

A collection of small Python utility functions for working with lists, tuples,
and strings.

## Functions

**`0-print_list_integer.py`**
Prints every integer in a list, one per line.

**`1-element_at.py`**
Returns the element at a given index in a list. Returns `None` if the index
is negative or out of range.

**`2-replace_in_list.py`**
Replaces the element at a given index in a list with a new value, and
returns the modified list. Leaves the list unchanged if the index is
invalid.

**`3-print_reversed_list_integer.py`**
Prints every integer in a list in reverse order, one per line.

**`4-new_in_list.py`**
Returns a new list with the element at a given index replaced, without
modifying the original list.

**`5-no_c.py`**
Removes every `c` and `C` character from a string and returns the result.

**`6-print_matrix_integer.py`**
Prints a matrix (a list of lists of integers), one row per line.

**`7-add_tuple.py`**
Adds two tuples element-wise and returns the result as a new tuple. Missing
values are treated as `0`.

**`8-multiple_returns.py`**
Returns a tuple containing the length of a string and its first character.
Returns `None` as the character if the string is empty.

**`9-max_integer.py`**
Returns the largest integer in a list, or `None` if the list is empty.

**`10-divisible_by_2.py`**
Returns a new list of booleans indicating which integers in the original
list are even.

**`11-delete_at.py`**
Returns a new list with the item at a given index removed. Leaves the list
unchanged if the index is invalid.

**`12-switch.py`**
Swaps the values of two variables.

## Usage

Each file can be imported as a module or run directly:

```bash
./0-print_list_integer.py
```
